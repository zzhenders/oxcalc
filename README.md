# oxcalc

Simple command-line programmer's calculator.

### usage:
  oxcalc [-h] [-b BASE] <expression>

### bases:
  he[x]adecimal, [d]ecimal, [b]inary, [c]haracters

### arithmetic operators:
  * a b +
  * a b -
  * a b *
  * a b /
  * a b %
        
### logical operators:
  * a b &
  * a b |
  * a ~
  * a b ^
  * a b <
  * a b >

*note: you may need to escape operators to avoid shell conflicts.*
